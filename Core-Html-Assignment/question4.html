1. <head> : Declares the document type and version of HTML being used.

  2. <title> : It defines the title of the web page.

  3. <meta> : It provide information about the web page that not shown in webpage.

  4. <link> : By this tag we can link CSS, JavaScript external file the the html.

  5. <style> : By using this tag we can style the html using CSS.

  6. <body> : All the content in body tag will be shown when rendered.

  7. <h1> : h1 tag is for the biggest heading. The are heading tag from h1 to h4 according to their size.

  8. <p> : p tag is use for rendering peragraphs in webpage.

  9. <a> : It creates hyperlinks in html.

  10. <img> : It use for rendering image in webpage.

  11. <ul> : It's use for making unordered list.

  12. <ol> : Use for making ordered list.

  13. <li> : It is use inside of ul or ol to define thelist items.

  14. <table> : table tag is use for making table in html.

  15. <tr> : It is used inside table tag for making table rows.

  16. <th> : It is for making table head inside tr.

  17. <td> : It is for making table data inside tr.
