Features Of HTML5

  • Video And Audio tags: By using this tag we can include videos and audio in the webpage.
  • Header: It helps to define the header of a webpage Header include logo, navbar, seach etc.
  • Footer: It is use for defining webpage footer which may contain footer logo, footer menu, copyrigth etc.
  • Semantic elements: Semantic elements are those which clearly describes it meaning to developer's and browsers. It also good for screen readers.
  • Email attribute: type="email" attribute is used in input tag. It specifies that the required input is email.
  • Nav tag: It uses for containing the nav links of a webpage. nav tag is good for accessibility. It also helps the screen readers to find the nav links.
  • Progress tag: Its create a progress bar in the webpage.
