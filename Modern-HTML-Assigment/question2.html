HTML Entities:

HTML Entities are codes for those characters (©, ™, <) which are either not included in keyboard or is not accurately visiable when typed.Thats why specific codes like &copy; made for these characters to use in html code and making these characters properly visiable in the webpage.

5 commonly used HTML entities:

  • &lt; for the character <
  • &gt; for the character >
  • &copy; for the character ©
  • &euro; for the character €
  • &trade; for the character ™
